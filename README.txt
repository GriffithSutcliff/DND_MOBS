1 Для запуска проекта установите node js с официального сайта.
2 Скачиваете репозиторий нажав на сайте гит кнопку code. Выбираете скачать zip.
3 Распаковываете где удобно, открывайте папку со всем проектом в vs code
4 откройте в vs code терминал и напишите npm install
5 напишите в терминале npm run start
уже запущенная версия: https://dnd-mobs.vercel.app/
